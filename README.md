# Facebook Login Page Clone

A simple **clone of the Facebook login page** built using **HTML, CSS and Tailwind CSS** for layout and styling.  
This project is designed for learning and practicing frontend UI skills.

---

## 🧾 Features

✔️ Responsive layout (mobile & desktop)  
✔️ Tailwind CSS based styling  
✔️ Clean and modern UI  
✔️ Recreated input fields and buttons  
✔️ Hover & focus effects
